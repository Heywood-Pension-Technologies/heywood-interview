#### Heywood Pension Technologies Developer Assessment

There are two parts to your assessment

1.  Coding challenge - around 50 minutes
2.  Traditional interview with Principal Developers - around 50 minutes

We'll make sure there's time for a quick break approximately half way through.
 
##### Prerequisistes

1. Somewhere quiet for up to two hours
2. An internet connected PC capable of running IntelliJ IDEA & Microsoft Teams
3. A webcam, microphone & speakers (or a headset)

##### Setup

Please carry out this well in advance of your interview slot.

1. Download and unzip the Amazon Corretto 17 from https://docs.aws.amazon.com/corretto/latest/corretto-17-ug/downloads-list.html
2. Set this JDK as JAVA_HOME
3. Install the latest IntelliJ IDEA community edition from https://www.jetbrains.com/idea/
4. Open this project in IntelliJ - it should automatically import
5. You should now be able to successfully run the DoesNothingTest

##### Just before your assessment

1. Open the project in IntelliJ IDEA and be prepared to share your screen
2. Join the Microsoft Teams Meeting in your invite
3. Good Luck!
